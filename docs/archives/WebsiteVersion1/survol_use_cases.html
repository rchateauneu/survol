SURVOL USE CASES 






This page summarizes very different use cases, on various platforms. They share a common scenario:
  • Choosing a running process or an application
  • Then trying to have as much information as possible, by trial and error.

The results are presented as commented images, and some hypothesis based on them. Several successive Survol versions might be used in these explorations, because the software evolves based on these use cases. Still, the core functionalities and general interface behaviour are stable.

  • Analysing Windows Explorer: This experiments Survol on a simple application with well-known behaviour.
  • Telnet connection from client to server: This analyses a simple IP (Telnet) connection between a Linux client and a Windows server.
