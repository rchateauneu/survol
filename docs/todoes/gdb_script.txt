Create gdb scripts to log activity in RDF format.

Writes RDF document to a HTTP server.
Possibly create SPQRL queries inserting RDF triples (This is a standard Api for a Sparql end-point).

Examples:
set logging file my_god_object.log

printf "D32: %Hf - D64: %Df - D128: %DDf\n",1.2345df,1.2E10dd,1.2E1dl
