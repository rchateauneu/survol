Survol Use Cases

This page summarizes very different use cases, on various platforms. They share a common scenario:

  • Choosing a running process or an application.
  • Then trying to have as much information as possible, by trial and error.

The results are presented as commented images, and some hypothesis based on them. Several successive Survol versions might be used in these explorations, because the software evolves based on these use cases. Still, the core functionalities and general interface behavior are stable.

  • Analysing Windows Explorer: this experiments Survol on a simple application with well-known behavior.
  • Telnet connection from client to server: This analyses a simple IP connection between two machines.
