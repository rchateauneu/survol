This page gathers various interbal or external urls for technical purpose, tests, documentaiton, information etc...
Javascript display Javascript entry point based on D3 library. It starts by displaying the tree of running processes on the server
Entry point Survol entry point: It displays in SVG mode the top-level scripts, each of them generates a specific set of data
Entry point in HTML Entry point of dynamic objects description pages, in HTML mode
Entry point in WMI object model This is the description page for a WMI object. An entity id must be given, so this link is just given as an information. Also, it can only work on a Windows platform because WMI is needed
Entry point in WBEM object model This is the description page for a WBEM object on the current WBEM server. An entity id must be given, so this link is just given as an information
Namespaces in WMI object model This is information only, as Survol only uses root/cimV2 namespace
Namespaces in WBEM object model This is information only, as Survol only uses root/cimV2 namespace
WMI portal Entry point to the current WMI server, if there is one (N/A on non-Windows machine if WMI is not present)
WBEM portal Entry point to the current WBEM server, if there is one (Linux only)
Survol classes hierarchy Display the complete hierarchy of classes on this Survol server
WMI classes hierarchy A top-level class must be given. This applies to Windows only
WBEM classes hierarchy A top-level class must be given. A WBEM server must be setup
Object classes Available types of objects available on this host: This is equivalent to the Survol ontology
WMI object types Complete hierarchy of WMI classes in the default namespace (root/CIMV2). Windows only.
WBEM object types Complete hierarchy of WBEM classes in the default namespace (root/CIMV2). A default WBEM server must be available.
RDF to Javascript text display This loads a RDF URL with RDFQuery and displays it with Javascript only
CGI variables Print CGI variables and various internal data. Development only
Bookmarks
Edit server configuration Display and edit the configuration of the current Survol server. Authorization needed.
SPARQL User interface Enter a SPARQL query.

Ontologies

WMI RDFSRDFS ontology extracted from Windows Management Instrumentation model. It contains hundreds of classes and thousands of properties
WBEM RDFSRDFS ontology extracted from the current WBEM server
Survol RDFSSurvol ontology, very light, focuses only on what is needed. It is compatible with WMI and WBEM ontologies, and just need to be merged to have more detailed information
