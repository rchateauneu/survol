This package pydbg intends to be compatible to the original pydbg at https://github.com/OpenRCE/pydbg

Its changes are partly merged to the fork https://github.com/rchateauneu/pydbg
and partly tested on Travis at https://travis-ci.org/github/rchateauneu/pydbg

The intention is to make local changes in this directory, merge them to rchateauneu/pydbg
where they are tested in Travis, then make a GIT pull request to OpenRCE/pydbg
